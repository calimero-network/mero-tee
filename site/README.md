# Interactive verification frontend

This folder contains a static frontend intended for GitHub Pages hosting.

- Entry point: `site/index.html`
- Assets: `site/styles.css`, `site/app.js`
- Deployment workflow: `.github/workflows/github-pages.yaml`

After enabling Pages for this repository, the site is expected at:

- `https://calimero-network.github.io/mero-tee/`

If your Pages base URL differs, use the URL from the `Deploy GitHub Pages` workflow output.
